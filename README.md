# LinkHub Premium

LinkHub Premium is a fully responsive, modern Linktree-style profile page. It features a **glassy design**, animated **grid background**, hover glow buttons, and **fully configurable per customer**. Perfect for personal profiles, businesses, or brands.

---

## Features

- Fully **responsive** on mobile, tablet, and desktop.
- **Glassy, frosted buttons** with hover glow animations.
- Animated **grid background** for a modern web aesthetic.
- **Semantic HTML**: `<header>`, `<section>`, `<footer>` for SEO and accessibility.
- **Dynamic configuration** via `script.js`.
- Customizable **colors**, buttons, profile info, and footer.
- Smooth animations for profile, buttons, and footer.
- Easy to **add/remove links** with icons (WhatsApp, Instagram, Location, Menu, etc.).
- Footer included, editable per client.
